# Gestiune-Farmacie-in-C
Aplicație care gestionează medicamentele unei farmacii. Creată în C. 

# Funcționalități
* Adăugare medicament
* Actualizare medicament
* Ștergere medicament
* Sortarea listei de medicamente după nume și cantitate (crescător/descrescător)
* Vizualizarea medicamentelor unde stocul disponibil e mai mic decât o valoare dată
* Vizualizarea medicamentelor unde numele medicamentelor începe cu o literă dată
* Vizualizarea medicamentelor unde concentrația este mai mare decât o valoate dată
* Printare listă medicamente
* Undo pentru lista de medicamente
* Exit
